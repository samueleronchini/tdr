Overview

The tdr tool estimates the targeted detectability range (D90) for compact-binary gravitational-wave signals associated with an external astronomical trigger.

Inputs include trigger time, localization (RA/DEC or sky map), SNR settings, and optional custom inclination interval.

Workflow

  1. Identify the LVK run from trigger time t0.
  2. Check public GWOSC data availability for H1/L1/V1.
  3. Download 256 s strain around the trigger.
  4. Estimate PSD for available detectors.
  5. Generate BNS and NSBH injections.
  6. Compute SNR and derive D90 targeted range.
  7. Write plots, maps, logs, and JSON summaries.

Installation

Prerequisites

  • Python 3.11
  • Scientific build toolchain (C/C++)
  • wget or curl in PATH
conda create -n TDR python=3.11 pip
conda activate TDR
pip install -r requirements.txt
pip install -e .

Quick start

Run with sky coordinates

tdr \
  --output-dir GRB_GIT \
  --t0 2020-03-26T12:24:47.903 \
  --ra 245.33 \
  --dec -21.08 \
  --snr-threshold 8.5 \
  --snr-type mf

Run with sky map

tdr \
  --output-dir GRB_GIT \
  --t0 2020-03-26T12:24:47.903 \
  --skymap-file examples/glg_healpix_all_bn200326517_v00.fit \
  --snr-threshold 8.5 \
  --snr-type mf

Command options

Option Description
--output-dir Directory where outputs are written.
--t0 Trigger time in GPS seconds or ISO UTC.
--ra, --dec Localization coordinates in degrees.
--skymap-file HEALPix FITS sky map localization.
--iota-min, --iota-max Custom inclination interval in degrees.
--snr-threshold D90 detection threshold (default 8.5).
--snr-type mf (matched-filter-like) or opt (optimal).

Mass combinations

BNS

  • m1=1.0, m2=1.0
  • m1=1.4, m2=1.4
  • m1=2.0, m2=2.0

NSBH

  • mBH=5.0, mNS=1.0
  • mBH=10.0, mNS=1.4
  • mBH=20.0, mNS=2.0

Notes

  • First run is slower because GWOSC strain files are downloaded.
  • Later runs are faster with cache reuse.
  • If GWOSC is temporarily unreachable, rerun later.
